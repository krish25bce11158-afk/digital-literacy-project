## Project Overview
This repository contains my Digital Literacy Portfolio created as part of the CSE0001 Digital Literacy course. The aim of this project is to understand how digital tools can be used responsibly, professionally, and safely in academic and real-life situations.

Through this project, I learned how to create a professional online presence, communicate effectively through email, use online collaboration tools, and understand cyber safety practices. Each task in this repository represents a practical skill required to become a responsible digital citizen.

## Repository Structure

### task-1-presentation
Contains the infographic created using Canva explaining Digital Literacy concepts such as safe internet practices, email etiquette, and digital tools useful for students.

### task-2-portfolio
Contains screenshots of my professional profiles including GitHub, LinkedIn, and Kaggle which represent my digital presence.

### task-3-platforms
Contains proof of completing a beginner coding challenge and screenshots of the Google Form created for Digital Literacy awareness quiz.

### task-4-email-etiquette
Contains two professional emails:
1. Request for assignment extension
2. Internship inquiry email

Also includes a checklist of social media do’s and don’ts.

### task-5-cybercrime
Contains a case study explaining a type of cybercrime and a checklist describing preventive safety measures students should follow online.

### report
Contains the final project report in PDF or Word format explaining all tasks and learnings.
